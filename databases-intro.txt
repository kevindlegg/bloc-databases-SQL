2. Explain when a database would be used. Explain when a text file would be used.
    - databases are commonly used when there are many users accessing/inputting data from many different endpoints
    - text files would be better suited when data is stored for a singular session and not shared/updated by many

3. Describe one difference between SQL and other programming languages.

4. In your own words, explain how the pieces of a database system fit together at a high level.
    - a database holds tables, a table holds rows, a row hold columns, columns hold data or like data types

5. Explain the meaning of table, row, column, and value.
    - table is the container for like data
    - row is an individual group of table data
    - column is a container of like information with a value
    - value is the data in each column

7. Given this payments table, provide an English description of the following queries and include their results:

     SELECT date, amount
     FROM payments;

        - retrieve the date and amount columns from the payments table

     SELECT amount
     FROM payments
     WHERE amount > 500;

        - retrieve the amount from the payments table where the amount is greater than 500

     SELECT *
     FROM payments
     WHERE payee = 'Mega Foods';
